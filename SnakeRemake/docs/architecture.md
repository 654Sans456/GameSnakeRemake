# Архитектура системы

**Основные компоненты:**
1. **MainWindow.xaml:**  
   - Определяет интерфейс: игровое поле (`Canvas`), текстовые блоки, меню.
   - Обрабатывает события клавиатуры (`Window_KeyDown`).

2. **MainWindow.xaml.cs:**  
   - Содержит логику игры: движение змейки, ботов, генерацию еды.
   - Классы: `MainWindow` (основная логика), `DispatcherTimer` (игровой цикл).

3. **Графические элементы:**
   - `Rectangle` для отрисовки змейки, ботов и еды.
   - Динамическое обновление позиций через `Canvas.SetLeft` и `Canvas.SetTop`.

4. **Обработка состояний:**
   - Стартовое меню (`StartMenu`), меню поражения (`GameOverMenu`).
   - Перезапуск игры через `RestartButton_Click`.
