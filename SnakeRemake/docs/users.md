# Пользователи и бизнес-процессы

**Целевая аудитория:**
- Игроки, интересующиеся ретро-играми с усложнённым геймплеем.
- Пользователи, желающие проверить свои навыки в условиях повышенной сложности.

**Бизнес-процессы:**
1. Запуск игры через стартовое меню.
2. Управление змейкой с помощью клавиатуры.
3. Сбор еды для увеличения счёта и длины змейки.
4. Избегание столкновений с врагами и границами поля.
5. Рестарт игры после поражения.
6. Завершение игры через меню или закрытие приложения.
