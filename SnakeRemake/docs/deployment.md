# Развертывание и настройка

**Установка:**
1. Убедитесь, что на системе установлен .NET Framework 4.7.2.
2. Скачайте и распакуйте архив с игрой.
3. Запустите файл `SnakeRemake.exe`.

**Настройка:**
- Параметры игры (скорость, сложность) заданы в коде и не требуют ручной корректировки.
- Для изменения размера поля или частоты появления ботов потребуется модификация исходного кода.
