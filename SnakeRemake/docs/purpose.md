# Назначение программы

**SnakeRemake** — это ремейк классической игры "Змейка" с расширенным функционалом.

Основные цели программы:
- Предоставить пользователю современную версию игры с элементами вызова.
- Добавить врагов двух типов: атакующих игрока, и конкурирующих за сбор еды.
- Реализовать систему подсчёта очков и динамическую генерацию новых противников.
- Создать интуитивно понятный интерфейс с меню старта, завершения игры и возможностью рестарта.
