# Системные (нефункциональные) требования

- **ОС:** Windows 7 или новее.
- **Платформа:** .NET Framework 4.7.2 или выше.
- **Процессор:** 1 ГГц или выше.
- **Оперативная память:** 512 МБ.
- **Видеокарта:** Поддержка DirectX 9.
- **Разрешение экрана:** Минимум 650x520 пикселей.
- **Дополнительно:** Клавиатура со стрелочками для управления.
